Standard Operating Procedure

 

  1. Connect the power cord and switch on the Main Switch
  2. Switch ON the equipment power switch.
  3. Enter the details of the sample in the equipment log book.
  4. Open the LID by pressing OPEN button.
  5. Fix a Suitable Rotor based on the required RPM (Swinging bucket rotor, Plate Rotor, Fixed Angle Rotor).
  6. Select the Appropriate model number of the rotor fixed in the equipment using rotor selection button.
  7. Set the temperature using the TEMP button.
  8. Set the desired time using the TIME Button.
  9. Set the Desired RPM/ RCF/ G-FORCE using the RPM or RCF Button. 
  10. Select the desired acceleration speed using ACCEL Button. 
  11. Select the desired deceleration speed using DECEL Button. 
  12. Keep the samples inside the ROTOR. 
  13. Ensure the samples are tightly closed to avoid spillage.
  14. Balance the samples with balancing tubes to avoid rotor imbalance. 
  15. Close the LID and wait till the temperature drops to the desired temperature if required.
  16. Press the START Button to centrifuge. 
  17. After the centrifugation is over you will hear a beep sound. 
  18. Then press the OPEN Button and take your samples without disturbing them. 
  19. Allow the equipment to defrost if the temperature is set below the ambient temperature. 
  20. Wipe the moisture present inside the rotor and the internal walls. 
  21. Ensure the internal environment is free from moisture. 
  22. Switch OFF the equipment switch.
  23. Switch OFF the main switch.
